VCS Implementationn Using Github

Create repository

Click on new repository.
Enter details.

Clone repository

Copy your repository path.
Move to directory where you want to clone your repository and then paste path in clone command.

Local repository

Now move to your local repository and start working in it.

Add file to local repository

Create a simple html file named "index.html" in your local repository.
" git add . " is used to add all changes in the working directory to the staging area.

Commit changes on local repository

Using the "git commit" command only saves a new commit object in the local Git repository.
You can also write commit message.

Push changes to remote repository

Push all the changes to remote repository.
Now you can see your index file is being added on remote repository.

Edit file on remote repository

Click on "pen" shaped icon to edit your file online.
